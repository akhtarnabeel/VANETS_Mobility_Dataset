"Metrics" contains code for genrating graphs for different performance metrics like 

Neighbour Distance Distirbution, 
Node Degree, 
Link Duration, 
No. of Clusters, 
Closeness Centrality,
Clustering Coefficient
and Size of Largest Cluster. 

The code can be run on 
any machine which has Matlab installed in it. 

*************************************************
"Mobility" contains code for generating figures related with mobility of vehicles 
like for generating graphs for 'Flow', 'Speed' and 'Error in flow and speed'. It also 
contains input files for the SUMO that can be used to generate traces for vehicles for 
low and high density traffic.
